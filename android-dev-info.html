My Android App

Details & Features

Overview

My Event Manager Android app is designed to help users efficiently organize, track, and manage their events and tasks. The app provides a streamlined experience for creating new events, setting reminders, and viewing upcoming activities in a clear and intuitive interface. Built with Java and Kotlin using Android Studio, the app leverages Material Design for a modern look and feel.

  • Event Creation: Users can quickly add new events with details such as title, description, date and time.
  • Reminders & Notifications: Integrated notification system ensures users never miss an important event or deadline.
  • Event List & Calendar View: Events are displayed in a list format, making it easy to visualize upcoming tasks.
  • Editing & Deleting: Users can edit or remove events as needed, keeping their schedule up to date.
  • Data Persistence: All event data is stored securely using local storage and can be synced with Firebase for backup and multi-device access.
  • User Authentication: Optional login system allows users to protect their event data and access it from multiple devices.
  • Accessibility: Designed with accessibility in mind, including support for screen readers and high-contrast modes.

Technologies Used

My Event Manager Android app is built using modern Android development tools and best practices to deliver a robust, user-friendly experience. The project demonstrates proficiency in both Kotlin and Java, following Material Design guidelines for a polished interface.

  • Android Studio: Main IDE for development, debugging, and UI design.
  • Kotlin & Java: Core programming languages for app logic and UI components.
  • Material Design: Ensures a modern, consistent, and accessible user interface with support for dark/light themes.
  • Fragments & Navigation Component: Modular UI and seamless navigation between screens.
  • RecyclerView & Custom Adapters: Efficiently displays and manages lists of events.
  • ViewBinding: Safe and efficient access to UI elements.
  • SharedPreferences: Persistent storage for user settings and preferences.
  • ItemTouchHelper & Swipe Gestures: Enables quick edit/delete actions on events.
  • AlertDialog: Provides intuitive event editing interfaces.
  • Vector Drawables & Material Icons: Scalable, consistent icons for improved UX.
  • BroadcastReceiver: Handles event reminders and notifications in the background.
